# Ikherbane Chaussettes — E-Commerce Website

## What This Is

An e-commerce website for **SARL Ikherbane Chaussettes**, an Algerian sock/textile manufacturer based in Bordj Bou Arreridj / El Eulma. The site clones the UI/UX design aesthetic of [carbonix-dz.com](https://carbonix-dz.com) (a Shopify-based car accessories store) but replaces all automobile content with Ikherbane's textile/sock products.

## Core Value

A premium, modern e-commerce showcase for Ikherbane socks — visually matching the clean, professional Carbonix design language while featuring Ikherbane's product categories and branding.

### Design Reference: Carbonix-dz.com
The UI/UX to replicate:
- **Navbar**: Dark/charcoal background, bold logo with red accent ("X"), nav links (Home, Catalog, Contact), right-side icons (search, account, cart)
- **Hero Section**: Full-width product imagery, scrolling marquee banner below
- **Collections Grid**: "Achetez par collection" — rounded cards with brand logos in 3-column responsive grid
- **Product Cards**: Clean grid with product images, titles, prices
- **Trust Badges**: 3-column section — fast delivery, COD payment, 24/7 support
- **Footer**: Dark background, centered large brand logo, description text, policy links, social media icons
- **Color Scheme**: Dark charcoal (#1D1D1D) + white (#FFFFFF) + red accent (#E12026)
- **Typography**: Clean sans-serif (Montserrat/Poppins style)
- **Overall**: Modern, clean, professional e-commerce
