Données Collectées

Nous collectons les données personnelles suivantes uniquement dans le cadre du traitement de vos commandes :

  • Nom complet — pour identifier votre commande
  • Numéro de téléphone — pour vous contacter concernant votre commande
  • Adresse de livraison — pour livrer votre commande dans la wilaya souhaitée
  • Wilaya — pour calculer les frais de livraison

Nous ne collectons pas volontairement votre adresse email via le site web. L'email peut être demandé uniquement lors de l'inscription à votre espace client.

Utilisation des Données

Vos données personnelles sont utilisées exclusivement pour :

  • Traitement des commandes — préparation et expédition de votre commande
  • Service client — réponse à vos questions et suivi de commande
  • Communication via WhatsApp — envoi des détails de commande via notre WhatsApp pour confirmer et traiter votre commande
  • Livraison — transmission de votre adresse à notre service de livraison

Vos données ne sont jamais vendues, louées ou transmises à des tiers à des fins marketing.

Stockage des Données (localStorage)

Notre site stocke certaines informations dans le localStorage de votre navigateur :

  • Contenu du panier — pour mémoriser les articles que vous avez ajoutés
  • Préférences utilisateur — pour améliorer votre expérience de navigation
  • Données de commande — pour faciliter le processus de commande

Ces données sont stockées uniquement sur votre appareil. Nous n'avons pas accès à ces informations depuis nos serveurs. Vous pouvez supprimer ces données en effaçant le cache de votre navigateur.

Partage via WhatsApp

Cette méthode nous permet de traiter votre commande rapidement et efficacement. En utilisant ce service, vous acceptez que vos données de commande soient transmises via WhatsApp.

Veuillez noter que WhatsApp a sa propre politique de confidentialité. Nous vous encourageons à la consulter.

Cookies

État actuel : Notre site n'utilise pas de cookies de traçage ou d'analytique actuellement.

Nous n'utilisons pas de Google Analytics ni d'autres services d'analytique pour le moment.

Future utilisation : Dans le cadre de l'amélioration de notre site et de l'expérience utilisateur, nous envisageons d'utiliser Google Analytics à l'avenir. Google Analytics utilise des cookies pour collecter des données statistiques anonymes sur la navigation des utilisateurs (pages visitées, temps passé, etc.).

Si nous activons Google Analytics, nous vous informerons via une bannière de consentement aux cookies et mettrons à jour cette politique de confidentialité.

Google Analytics Free Tier permet jusqu'à 200 emails par mois pour ses rapports standards. Cette limite ne concerne pas les visiteurs du site, mais les envois de rapports par email.
